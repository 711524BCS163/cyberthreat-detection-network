# Cyber Threat Detection Network

A compact, educational C++17 project that implements multiple data structures for threat detection and logging. This repository provides sources, unit tests, examples, and CI configuration to get you started quickly.

[![Build](.github/workflows/build.yml/badge.svg)](https://github.com/)

## Quick start

1. mkdir build && cd build
2. cmake -DBUILD_TESTS=ON ..
3. cmake --build . --config Release
4. ./bin/threat_detection_system --demo

See `docs/SETUP_GUIDE.md` for full details.
