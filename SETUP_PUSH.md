# Push to GitHub

1. Create a repository on GitHub (e.g. `cyber-threat-detection-network`).
2. On local machine:

```bash
cd path/to/cyber-threat-detection-network
git init
git add .
git commit -m "Initial commit: Cyber Threat Detection Network"
git remote add origin https://github.com/YOUR_USERNAME/cyber-threat-detection-network.git
git branch -M main
git push -u origin main
```

Replace `YOUR_USERNAME` with your GitHub username.
